In this step, you'll install the `zrok2` command-line tool on your device.

:::note
zrok is not supported on mobile devices.
:::

Select your operating system below. Click **GUIDE** for step-by-step installation instructions, or use the
architecture links to download the binary directly. If you're not sure which to pick, use the guide.
