## Install the binary

Download the binary distribution for your macOS architecture. For Intel Macs use the `amd64` distribution. For Apple
Silicon Macs use the `arm64` distribution.

1. Unarchive the distribution in a temporary directory:

    ```bash
    cd ~/Downloads && mkdir -p /tmp/zrok2 && tar -xf ./zrok*darwin*.tar.gz -C /tmp/zrok2
    ```

1. Install the `zrok2` executable:

    ```bash
    mkdir -p ~/bin && install /tmp/zrok2/zrok2 ~/bin/
    ```

1. Add `~/bin` to your shell's executable search path:

    ```bash
    PATH=~/bin:$PATH
    ```

    To persist this across sessions, add the line above to your `~/.zshenv`.

1. Verify the install:

    ```bash
    zrok2 version
    ```
